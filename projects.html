Retro Arcade Box

This is a bartop arcade box that I built and programmed using MDF, a Linux OS, and emulators. With over 700 titles installed, the only thing lacking from the box is a coat of paint. This box will play N64 games all the way back to the classic Pac-Man, Galaga, or Frogger.
